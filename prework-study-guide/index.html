HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • 1/3 core languages of the web

CSS

  • Language to describe the presentation of a website thats created with HTML and how each element on a webpage is renendered
  • Controls the layout, colours, fonts, and overall appearance of a webpage
  • 2/3 core languages of the web

Git

  • A distributed version control system for tracking changes in code files
  • Enables collaboration among developers by managing code versions, brances, and merges
  • Provides tools for code sharing, conflict resolution, and code review in software development projects
  • it status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A programming language used to provide logic and interactivity for websites
  • Widely used for creating animations, validating user interactions, and performing calculations
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • 3/3 core languages of the web
